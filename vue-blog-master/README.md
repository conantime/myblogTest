## 前端项目结构

```
|-src  					//开发
|----App.vue  			//背景初始化
|----main.js  			//初始化各种配置
|----router.js  		//路由
|----assets  			//资源
|----plugin       //一些公共方法
|----components  		//组件
|----utils  			//axios——http插件
|----vuex  				//状态管理
```

## 启动

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
